현재 차원의 날씨를 확인할 수 있습니다.

<br/>

예를 들어, 현재 날씨가 비인지 확인하는 구문을 작성해 보겠습니다.

`is_raining.json`
```json
{
   "condition": "minecraft:weather_check",
   "raining": true
}
```

```mcfunction
test:a
   execute as @a[predicate=test:is_raining] run say Rain!
```
