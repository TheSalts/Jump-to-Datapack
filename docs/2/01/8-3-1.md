마인크래프트에서 `scoreboard`는 게임 내 다양한 값을 정수 형태로 저장하고 사용할 수 있으며, 주로 플레이어나 엔티티의 특정 상태 및 행동을 기록하고 변수로 활용됩니다.

이 점수(`score`)는 `-2³¹` ~ `-2³¹-1`의 범위 내에서 마음대로 조정(읽기만 가능한 조건 존재)할 수 있고, 세계 폴더의 `data/scoreboard.dat` 파일에 저장됩니다.

<br/>

`/scoreboard` 명령어의 `<target/targets>` 인수 부분에는 [[:paper: 01-3 선택자](#2/01-3)]뿐만 아니라, 일반 문자(띄어쓰기 불가)를 사용할 수 있습니다.

예를 들어, 다음 명령어를 입력하여 `a`의 `test` 점수판 점수를 1로 설정할 수 있으며,
```mcfunction
/scoreboard players set a test 1
```

일반 문자에는 한글도 포함되므로 다음과 같은 명령어도 실행 가능합니다.
```mcfunction
/scoreboard players set 한글 test 1
```

이로써 하나의 점수판에서 변수 여러 개를 쓰는 것처럼 `a`, `한글` 등 이를 통틀어 거짓 플레이어라고 부릅니다.

이 거짓 플레이어는 `execute if score` 명령어에서 똑같이 감지할 수 있습니다.

<br/>

특정 접두사에 따라 기능이 조금씩 달라지는데, 일반 문자 외에도 `#`(해시태그)으로 시작하는 문자와 `*`(와일드카드)가 있습니다.

`#`으로 시작하는 문자 같은 경우에는 다른 일반 문자 및 선택자와는 다르게 적은 비용을 소모하므로 성능 면에서 훨씬 이득입니다. 또한, `sidebar`에 표시되지 않는다는 특징을 지니고 있습니다.

```mcfunction
/scoreboard players set #a test 1
```

일반 문자가 `#`으로 시작하는 문자보다 비용이 상대적으로 많은 이유는 해당 서버의 모든 플레이어의 이름과 일반 문자가 같은지 확인하는 과정을 한 번 더 거치기 때문입니다.  
이는 서버의 플레이어 수에 비례하므로 `O(N)`의 시간 복잡도를 가집니다.

<br/>

`*`는 단독으로는 점수를 저장할 수 없으며 `reset`/`set` 등 `2`개 이상의 항목에 대해 모든 거짓 플레이어에게 적용됩니다.

예를 들어, `test` 점수판의 모든 점수 보유자를 초기화 하는 명령어는 다음과 같습니다.
```mcfunction
/scoreboard players reset * test
```

`*`는 `text_component_format`에 사용될 수 있습니다.  
예를 들어, `test` 점수판의 점수를 보유한 모든 플레이어의 `actionbar`에 본인의 점수를 표시하고 싶다면, 다음 명령어를 사용하면 됩니다.
```mcfunction
title @a actionbar {score:{name:"*", objective:"test"}}
```

이는 `execute as`를 이용하는 것보다 상대적으로 적은 비용을 소모합니다.
