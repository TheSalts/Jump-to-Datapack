`UUID`는 마인크래프트에서 128 비트의 숫자로 제공되는 고유 식별자입니다.  
대표적으로 모든 엔티티의 `UUID` nbt(`int[]`)에서 확인할 수 있습니다.

## 형식
- `-`(하이픈)으로 구분되는 16진수
- 16진수
- 정수 배열

`UUID`의 범위는 `-2¹²⁷` ~ `-2¹²⁷-1`로 모든 정수 값을 지원하며 값이 무작위로 지정되는데, 마인크래프트에서는 `version 4`, `variant 1`을 사용합니다.

<br/>

## 지정
즉, 특정한 비트를 제외한 전체 숫자가 무작위로 지정됩니다.
- `xxxxxxxx-xxxx-Axxx-Bxxx-xxxxxxxxxxxx`

A의 경우: 버전 4이므로 `4`로 고정  
B의 경우: `8, 9, A, B` 중 하나(각 확률은 `1/4`)  
x의 경우: 완전 무작위(`0[0]` ~ `15[f]`)

[[UUID converter](https://www.soltoder.com/mc-uuid-converter/)]에서 다음 작업을 수행할 수 있습니다.
- 무작위 `UUID` 생성
- 16진수 -> Int[]
- Int[] -> 16진수
- 플레이어 이름 -> 16진수
- 플레이어 이름 -> Int[]

이미 생성한 UUID와 새로 생성할 UUID가 서로 일치할 확률은 `0.00000000000000000000000000000000000027%`

<img class="latex" src="https://latex.codecogs.com/svg.image?\large 2^{-122} \approx 2.7 \times 10^{-37}" style="vertical-align:middle; filter: invert(1)"/> 입니다.

## 엔티티
마인크래프트의 모든 엔티티는 `UUID` nbt를 지니고 있으며 또한, 중복되지 않습니다.

이미 소환된 엔티티의 `UUID`는 변경이 불가능합니다.  
`EntityDataAccessor::setData()`
```java
UUID uuid = this.entity.getUUID(); // 변경 전의 UUID를 저장합니다.
...
try {
    this.entity.load(...) // 엔티티 nbt를 설정합니다.
    this.entity.setUUID(uuid); // 엔티티의 UUID는 고정입니다.
}
```

다만 `/summon` 명령어 사용 시, `UUID` nbt를 수정하여 원하는 `UUID`로 지정해 소환할 수 있습니다.

예를 들어, `16진수 하이픈` 값이 `00000000-0000-0000-0000-000000000000` = `0-0-0-0-0`인 표지를 소환하려면 다음 명령어를 입력합니다.
```mcfunction
/summon marker 0 0 0 {UUID:[I;0,0,0,0]}
```

이 `16진수 하이픈` 값은 선택자 자리에도 사용할 수 있으므로 다음과 같이 사용할 수 있습니다.
```mcfunction
/tellraw @a {selector:"0-0-0-0-0"}
```
```mcfunction
/say 0-0-0-0-0 표지.
```

이는 `@e[limit=1, nbt={UUID:[I;0,0,0,0]}]`과 동일하지만 죽어가는 엔티티를 선택할 수 있으며, [[:paper: 01-3 선택자](#2/01-3)]와 달리 시간복잡도가 `O(1)`이므로 비용면에서 매우 이득입니다.

<br/>

현재 차원이 아닌 차원에서는 똑같은 `UUID`를 가진 엔티티를 소환할 수 있습니다. [[:mojang: MC-184268](https://bugs.mojang.com/browse/MC-184268)]
```mcfunction
/execute in minecraft:overworld run summon marker 0 0 0 {UUID:[I;0,0,0,0]}

/execute in minecraft:nether run summon marker 0 0 0 {UUID:[I;0,0,0,0]}
```
이때, `UUID`를 선택자에서 사용하면 가장 처음 로딩되는 차원순으로 표지가 선택됩니다.
- 오버월드 -> 네더 -> 엔드
```mcfunction
# 표지가 minecraft:overworld, minecraft:nether에 소환되어 있다고 가정

/say 0-0-0-0-0 표지.
>>> minecraft:overworld의 표지가 선택됨.
```
