스토리지(`storage`)는 세계의 `맵/data` 폴더에 저장되는 [[gzip](https://en.wikipedia.org/wiki/Gzip)]  
[[nbt](https://minecraft.wiki/w/NBT_format)] 파일 중 하나로, 자유롭게 `nbt`를 저장 및 읽을 수 있으며,  
[[:paper: 01-5 Resource Location](#2/01-5)]으로 구분됩니다.

<br/>

스토리지는 `/data` 명령어로 `nbt`를 쓰거나, 읽을 수 있습니다.  
## 읽기
- `/data get`
- `/execute if|unless data`
- `nbt json format`

## 쓰기
- `/data modify storage`
- `/data merge storage`
- `/data remove storage`
- `/execute store`

## 경로
스토리지는 `월드/data/command_storage_<namespace>.dat` 파일에 저장됩니다.  
하나의 세계에는 여러 개의 스토리지 파일이 있을 수 있습니다.

일반적으로 `<namespace>`에 아무 것도 작성하지 않으면,  
`command_storage_minecraft.dat` 파일에 저장됩니다.
- `/data modify storage test a set value "test"`
- `/data modify storage a b set value 1`

한 개의 파일에 많은 양의 데이터가 저장되어 있으면,  
접근 시 상당한 지연이 발생합니다.
- `/data modify storage list l set value [0,1,2, ... 2147483647]`

따라서, `<namespace>`를 변경하여, 서로 다른 파일이 생성되게 해야 합니다.
- `/data modify storage a: b set value 1`
   - `command_storage_a.dat` 파일에 저장됨

- `/data modify storage test: string set value "test"`
   - `command_storage_test.dat` 파일에 저장됨
