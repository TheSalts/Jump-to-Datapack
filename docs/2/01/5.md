`Resource Location`은 `namespace:path` 혹은 **식별자** 라고도 불리며,  
마인크래프트에서 객체를 선언하고 충돌없이 구분하기 위해 사용하는 형식입니다.

마인크래프트에서 블록, 아이템, 엔티티, 함수([[:paper: 02-1 함수란?](#1/02-1)]) 등  
기타 다양한 객체를 참조하는 일종의 문자열입니다.

# 문자
다음 문자열은 식별자에 사용할 수 있습니다.
- `0~9`
- `a-z`
- `_`(under\_score)
- `-`(dash)
- `.`(dot)

다음 문자열은 `path`에서만 사용할 수 있습니다.
- `/`

# 문자열
`:` 문자를 경계로, 네임스페이스와 경로로 구분됩니다.
| namespace | path | string |
|-----------|------|--------|
| minecraft | diamond | `minecraft:diamond` |
| foo | bar | `foo:bar` |
| undertale | sans | `undertale:sans` |
