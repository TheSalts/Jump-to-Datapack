여러 개의 문자가 주어졌을 때, 그것을 하나의 단일 문자로 합치는 기술입니다.  
예를 들어, 다음 3개의 문자를 하나의 문자로 합칩니다.
- `"s"` + `"t"` + `"r"` -> `"str"`

# /enchant 사용

---

`/enchant` 명령어를 LivingEntity가 아닌 엔티티에게 사용하면, 이름이 평탄화되어 출력되는 버그를 이용합니다.
- [[:mojang: MC-170355](https://bugs.mojang.com/browse/MC/issues/MC-170355)]

반드시 명령 블록을 사용해야합니다.

## 구현
1. 엔티티를 이용하기 때문에 특정 청크를 로드합니다.
   - `/forceload add 0 0`
2. LivingEntity가 아닌 엔티티를 소환합니다.
   - `/summon marker 0 0 0 {CustomName:[s,t,r,i,n,g], UUID:[I; 0,0,0,0]}`
3. 명령 블록으로 `/enchant` 명령어를 실행합니다.
   - `enchant 0-0-0-0-0 lure`
4. 명령 블록의 `LastOutput`을 확인하면 `"string"`으로 변경된 것을 알 수 있습니다.
   - `{extra:[{color:"red",extra:[{translate:"commands.enchant.failed.entity",with:["string"]}],text:""}],text:"[HH:mm:ss] "}`
   - 경로: `LastOutput.extra[].extra[].with[]`

```mcfunction
forceload add 0 0
summon marker 8. 0 8. {UUID:[I; 0,0,0,0]}
setblock 0 0 0 oak_sign
setblock 0 1 0 command_block
```

```mcfunction
data modify entity 0-0-0-0-0 CustomName set value [s,t,r,i,n,g]
data modify block 0 0 0 front_text.messages[0] set value {nbt:"CustomName",entity:"0-0-0-0-0"}
data modify entity 0-0-0-0-0 CustomName set from block 0 0 0 front_text.messages[0]

# 명령 블록으로 실행합니다.
enchant 0-0-0-0-0 lure

data modify storage output: string set from block 0 1 0 LastOutput.extra[].extra[].with[]
tellraw @a {nbt:"string", storage:"output:"}
```
