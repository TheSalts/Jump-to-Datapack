마인크래프트에는 실행 대상을 변경할 수 있는 명령어가 존재하는데, 현재 명령어를 실행하는 대상인 자기 자신을 제외하는 명령어(혹은 인수)는 존재하지 않습니다.

이때 본인을 제외하는 방법 중 하나로 `/tag`를 이용합니다.
- 비용이 적고 간편합니다.

## 함수의 실행 순서
---
함수는 작성된 명령어를 위에서부터 아래 순서대로 실행합니다.
```mcfunction
say 1
say 2
tellraw @a "3"
```
> \>\>\> [Server] 1  
> \>\>\> [Server] 2  
> \>\>\> 3

함수를 실행하는 명령어(`/function`) 또한, 순서대로 실행합니다.  
그 함수와 연결된 다른 모든 함수도 실행합니다.
```mcfunction
say 1
function <second>
say 3

# second.mcfunction
   say 2
   tellraw @a "중도 실행"
```
> \>\>\> [Server] 1  
> \>\>\> [Server] 2  
> \>\>\> 중도 실행  
> \>\>\> [Server] 3

<br/>

## 구현
---
이러한 순차적 실행의 원리를 이용하여 자신에게 임시로 태그를 부여한 후, 곧바로 제거하면 본인을 제외시킬 수 있습니다.
```mcfunction
tag @s add this
   say @e[tag=!this]
tag @s remove this
```

1. `this` 태그를 임시로 부여합니다.
2. `this` 태그가 없는 엔티티를 대상으로 명령어를 실행합니다.
3. 임시로 부여했던 `this` 태그를 제거합니다.

<br/>

## 활용
---
- 히트스캔 구현 시, 사수를 제외할 수 있음
- 자신은 제외하고 모든 플레이어에게 메시지를 전송할 수 있음
- 특정 개체 주위를 탐색하되, 본인은 제외할 수 있음
