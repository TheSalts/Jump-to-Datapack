초시계(`/stopwatch`)는 기존의 월드보더 실시간 버그 [[:mojang: MC-53491](https://bugs.mojang.com/browse/MC/issues/MC-53491)]를 수정하면서 [[:paper: 03-2 월드보더 타이머](#2/03-2)] 기술을 대신해 [[:snapshot: <span style="font-family:Minecraft-Event"/>Minecraft 25w41a](https://minecraft.wiki/w/Java_Edition_25w41a)(1.21.11 스냅샷)]에 추가된 기능입니다.

# syntax
- `stopwatch create <id>`
- `stopwatch query <id> scale`
- `stopwatch restart <id>`
- `stopwatch remove <id>`

`<create>`: <id>의 초시계를 생성합니다.    
`<query>`: <id>의 현재 시간을 `[scale]`을 곱해 추적합니다.  
`<restart>`: <id>의 시간을 초기화합니다.  
`<remove>`: <id> 초시계를 제거합니다.

<br/>

<details class="details">
<summary><span/>초시계 <code>query</code> 시, 반환 값은 <code>int</code>이기 때문에 출력 창에 표시되는 시간과의 차이가 날 수 있습니다.</summary>

   ```java
   long i = Stopwatches.currentTime(); // 현재 밀리초를 가져옵니다.
   double d1 = stopwatch.elapsedSeconds(i); // 초시계의 현재 시간에 첫 실행 시간을 감산합니다.
   commandsourcestack.sendSuccess(() -> {
      return Component.translatable("commands.stopwatch.query", Component.translationArg(identifier), d1); // 출력 창에서는 소수점 이하 3자리까지 출력
   }, true);
   return (int)(d1 * d0); // 반환 값은 int
   ```
